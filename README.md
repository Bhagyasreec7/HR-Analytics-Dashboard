# HR Analytics Interative Dashboard creation using (MS Excel and Power BI)
## Project Objectives
Analyze employee attrition across different departments.
Monitor workforce demographics and employee distribution.
Identify high-risk employee groups.
Analyze attrition based on salary, gender, age, and experience.
Provide HR managers with data-driven insights for decision-making.

## Dataset Used
- <a href="https://github.com/Bhagyasreec7/HR-Analytics-Dashboard/blob/main/HR%20Analytics.csv">Dataset</a>

##Question(KPIs)
- What is the total number of employees?
- How many employees have left the organization?
- How many employees are currently active?
- What is the overall employee attrition rate?
- What is the average age of employees?
- What is the average years of experience of employees?
-<a href="https://github.com/Bhagyasreec7/HR-Analytics-Dashboard/blob/main/Dashboard%20img.png">Dashboard</a>

## Project Process
- Imported the HR Analytics dataset into Power BI.
- Cleaned and transformed the data using Power Query by removing duplicates, handling missing values, and correcting data types.
- Created DAX measures and KPIs to analyze employee attrition and workforce metrics.
- Built an interactive dashboard with visualizations to generate actionable HR insights and support data-driven decision-making.
